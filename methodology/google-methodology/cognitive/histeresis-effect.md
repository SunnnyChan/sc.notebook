# J曲线和滞后效应

## J曲线
```md
是一个政治学或者经济学上的规律。
```
* 经济上
```md
是指一国货币贬值或升值时，该国贸易收支及经常帐户收支状况一般并不能立即改善或恶化，往往要经过一段时间。

私募股权基金的内部回报率通常呈现J曲线效应，即正回报通常出现在基金寿命的后几年。
基金存续的早期阶段，私募股权基金有价值下降的趋势，这一阶段通常称作“眼泪的山谷”。
```
* 政治上
```md
美国政治学家尹恩•布雷默在研究了很多国家从落后走向繁荣、稳定的道理之后，同样发现了J曲线规律。
一个落后的国家如果按照现在最舒服的状况是没有希望的。

如果有幸找到了一条合适的发展道路，最开始一段时间可能国家的发展比现在的状况还要差，
但如果坚持正确的方向，过了一个坎，以后的发展就和最开始预设的方向一致了。
而且只要方向对，你走得越远，结果就会越好。

根据布雷默的研究，你看到曲线中间最低，右边比左右高，而且没有尽头，
也就是说改变现状，坚持好的防线，会一本万利，长期收益。
而且据他的研究和观察，没有一个国家逃过这个J曲线的命运。
凡是发展停滞不前的国家都是左右摇摆，遇到一点困难就缩回去，譬如俄罗斯。
```

* 个人学习知识和技能
```md
如果我们相信一个新的方法是好的，就坚持一段时间再下结论，不要根据实时的反应草率给出结论。
```

## 滞后效应
```md
要改变原先的状态，通常不能立竿见影，其效果总是滞后的。

有些时候，我们不知道要往前走多远才能看到曙光，对此的办法就是 慢三拍。
决定方向时，不妨慢一点，而一旦决定，更改之前，更要慢。
人越年轻越容易纠结自己该往什么方向努力，其实对于年轻人来讲，坚持一个方向足够长时间都会有效果。

凡是大的改动都要一点点来，因为改动的初期，可能都会有相反的结果，
如果改动太大，短期产生的反作用无法承受，就等不到J曲线上升的那一天了。
```
* 滞后效应对广告效果的影响
```md
是传播学上一个重要课题。

Google广告系统有两种：
一种是点击之后直接转换为购买，称为即时效应广告，比如电商广告，只要一定数量的广告，马上立竿见影。
但这种广告的累积效果比较差，只要竞争对手做同样的广告，流量就被引走了。

一种是品牌展示广告，广告效果具有滞后性，顾客从看到广告到觉得购买的过程可能要几周。
当他们对品牌产生认知、记忆、兴趣后，才会在心理上产生购买欲望。
但习惯一旦形成，会在一定时间内保持忠诚度，这就是潜移默化之后产生的累积效应。

Google 通常建议广告商合理的分配两种广告的预算。
```

## 吴军老师的总结
```md
1. 很多时候我们不能太心急，采取了行动，不要马上下结论，
  要等一段时间，等滞后效应出现了，看效果再下结论。
2. 人和人的任何长期关系，都是要对利益进行公平的分配，不可能总是一方占便宜。
3. 人还是要有所敬畏，这样可以让自己少犯错误。
```

## 一些思考
```md
大数据流分析，一般用于处理实时场景的分析，由于效果的滞后性，某些场景下应该要考虑滞后性对分析结果的影响。
```

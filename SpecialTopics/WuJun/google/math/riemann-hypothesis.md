# 黎曼猜想

```md
关于认识的延拓问题请先阅读 数的发展。
```
* 黎曼
```md
黎曼是19世纪德国著名的数学家，他奠定了非欧几何分支的黎曼几何的基础。
爱因斯坦的理论就是建立在黎曼几何基础之上的，此外他对微积分的公理化有很大的贡献。
```

* 调和级数
```md
所谓级数是将一个无穷的数列求和。
比如：等比级数 1 + 1/2 + 1/4 + 1/8 + ......

而调和级数是正整数的倒数和：
Z = 1/1 + 1/2 + 1/3 + 1/4 + ......
调和级数是发散的，和值为无穷大。

后来欧拉把调和级数稍作改变，改成正整数倒数平方和：
Z(2) = 1/1^2 + 1/2^2 + 1/3^2 + 1/4^2 + ......
欧拉告诉大家，S(2)等于一个有限的数值，即该级数是收敛的。
欧拉算出它等于 π^2/6，结果是对的，但欧拉的算法今天来看其实不严谨，
因为他首先需要证明数列是收敛的，他的做法才有意义。
但欧拉确实找到了一种计算这一类级数的方法。
```
* Zeta 函数
```md
把上面的级数再做一个推广：
Z(s) = 1/1^s + 1/2^s + 1/3^s + 1/4^s + ......
即整数倒数的s次方之和，s 可以为任何的数，这个级数在数学上被称为Zeta函数。

欧拉发现，只要s大于1，它就是收敛的，而小于1是发散的。
```
* 欧拉的思考
```md
当s为负数时，上面的级数会是什么样的？
这在现实世界中是一个很无趣的问题，因为它加来加去结果无非是无穷大。
比如 ： s = -1，级数就是 1 + 2 + + 3 + 4 + ......即正整数之和。

欧拉做了一个大胆的假设，如果依然使用收敛级数求和的方法来计算它，就会得到一个荒唐的结果：
S(-1) = 1 + 2 + + 3 + 4 + ..... = - 1 / 12
产生这种错误结论的原因是，用对收敛级数求和的方法计算不收敛的级数。
实际上不能将有限世界的方法，简单地应用到无限世界中。

欧拉还用通过的错误方法得到了另一个荒唐的结论，即所有正整数的平方和等于0。

上面的计算方式，在假设级数发散时可以用级数收敛时的计算方法，就会得到这样的结果。
也就是说，一旦设定了前提，不论通过什么逻辑得到什么结果，在数学上都行得通，
这是数学和自然科学的本质差别。
```
* Zeta 函数的延拓
```md
为了让这样没意义的结论变得有意义，欧拉做了一次函数定义的延拓，
即把本定义在s必须大于1基础上的 Zeta 函数Z(s)，定义的范围（也称为定义域）扩展到了所有的复数范围。

对于Z(-2)这样的函数值，不能让它等于无穷大，那样没有意义，就定义为欧拉的解法得到的结果。
给定s不同的数值，Zeta 函数Z(s)就能算出不同的值。
```
* 平凡解
```md
对于某些 s 的值，比如说 -2，-4，-6等等，Zeta函数的值恰巧是0。
于是这些特定的s 就被称为 黎曼方程Z(s) = 0的解，可以发现 s等于负偶数都是方程的解。
由于这些解很快被大家发现了，因此被称为平凡解。
```
* 什么是 黎曼猜想
> 那么除了 负偶数的解，黎曼方程是否还有其他的解呢？
```md
1859年，黎曼在当选为柏林科学院通信院士时，作为对这个崇高荣誉的答谢，
向柏林科学院提交了一篇“论小于给定数值的素数个数”的论文。

在论文中指出，黎曼方程的非平凡解不仅存在，而且都集中在复数平面的一条直线上，
当时他没有证明，因此这个结论就成了一个猜想。
```
* 黎曼猜想证明
```md
后来人们根绝黎曼的提示找到了很多非平凡解，它们都具有 1/2 + yi这样的形式。
其中i是-1的平方根，y是某个特定的数字。

由于Zeta 函数方程的解和素数的分布非常相关，因此随后的一百多年，有很多数学家研究这个课题，
并且发现了15亿个非平凡解，而且最大的一个平凡解，数值本身已经很巨大了。

实际上，只要愿意让计算机不断的计算下去，可以不断得到新的非平凡解，而且这些解符合黎曼的假设。
可以将，在我们能够搜索到的非常大的空间里，到目前为止能够找到的所有的解，都符合黎曼猜想，没有例外。

既然我们测试了很大的范围，能否认为现实世界中，黎曼猜想就是成立的，
因此不需要再考虑它在数学上的正确性了？
一方面，在工程和应用科学上，我们确实在使用还没有证实的猜想。
比如说我们今天相信各种加密系统是安全的，那只是从工程的角度讲，
从科学的角度讲，目前使用的各类加密方法都是能破解的，那只是时间的问题而已。
```

> 总结
* * 数学同实验科学和工程上的区别
```md
区别不仅仅在于结果和精度上，更关键在于解决问题的思维方式上。

采用纯数学的思维常常解决不了实验科学和工程上的问题，反之亦然。
如果把这种差异放大到各个领域，就会发现不同领域的工作，常常需要有不同的思维方式。
```
* * 关于 “钻牛角尖”
```md
实际上有一大类钻牛角尖现象的产生，是因为把合适他们工作和生活的思维方式，用错了场景。

但凡一个职业人士，都会不自觉地犯这一类“钻牛角尖”的毛病，自己认为很严谨的专业做法和说法，
其他职业者看来，就是钻牛角尖，因为后者希望对那些自己不熟悉的专业概念的理解模糊点、粗糙点。

为什么学理工科的人更容易给人钻牛角的看法？
因为他们自己觉得很准确的思维方式，在别人看来完全不需要。因此，看人讲话是一件很重要的事情。
```

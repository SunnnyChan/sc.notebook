# ECRS
```md
取消（Eliminate）
    “完成了什么?是否必要?为什么?”
合并（Combine）
    如果工作或动作不能取消，则考虑能否可与其他工作合并。
重排（Rearrange）
	如上所述，改变一下顺序，改变一下工艺就能提高效率。
简化（Simplify）
	将复杂的工艺变得简单一点，也能提高效率。
```

```md
无论对何种工作、工序、动作、布局、时间、地点等，
都可以运用取消、合并、改变和简化四种技巧进行分析，形成一个新的人、物、场所结合的新概念和新方法。
```
## 扩展ECRSI法
* 增加I（Increase）
```md
现有工序的基础上增加新的工序，来提高产品质量、增加产品功能，或者为后续工作做准备等
```

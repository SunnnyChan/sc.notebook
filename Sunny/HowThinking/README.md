# 怎样思路？


## 思维
```md
思维是人类在接收到外界输入信息后
	经过大脑有意识地处理后，形成有价值的输出结果的过程。
大脑会对接收到的信息进行 降噪，加工和模式匹配处理
	在处理的过程中我们需要借助我们大脑里面 已有的基础记忆，知识和经验信息 
	即外在输入+内在知识经验共同进行处理加工和模式匹配后形成 有价值的输出
```
